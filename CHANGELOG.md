# Changelog

## [1.0.2] - 2026-03-09
- Updated User Guide with HDICT links and usage instructions.
- Made HDICT links clickable in the footer and manual screen.
- Updated app version to 1.0.2+3 across all platforms (iOS, macOS, Android, Windows, Linux).
- Fixed `isar_flutter_libs` compile error in Android.

## [1.0.1] - 2026-03-01
- Made refresh button more prominent with "last updated" timestamp.
- Added download statistics to "Your Lists".
- Displayed statistics for available dictionaries in "Indic-dict".
- Added a "Stop" bar/button to cancel active downloads.
- Improved update detection for "Indic-dict" based on filenames with timestamps.
- Enhanced storage checking logic.
- Fixed `Info.plist` errors.

## [1.0.0] - 2026-02-27
- First stable release.
- Added app logo to the "About Us" page.
- Added application icon.
- Updated User Guide.
- Implemented one-pager layout.
- Renamed app to "Dictionary Updater".
- Ensured iOS build success.
- Improved macOS display of downloads directory (user-friendly paths).
- Standardized downloads to default locations for better reliability.
- Added support for web downloads.
- Implemented user manual updates.
- Added tests for "Indic-dict" update logic.
- Improved UI with app drawer instead of navigation rails.
- Removed unused imports and addressed analyzer issues.

## [0.1.2] - 2026-02-27
- Large download confirmation dialog.
- Left panel accessible at all times.
- Last sync time shown globally.
- Automatic loading of "Indic-dict" repository.
- Support for multiple archive formats (.7z, .rar, etc.).
- Categorized downloads by source name.
- Initial support for "Indic-dict" downloader.
